\doxysubsection{Detailed Description}
Represents an in-\/memory input byte stream. 

\mbox{\hyperlink{class_this}{This}} class is mainly for being wrapped by \mbox{\hyperlink{class_encoded_input_stream}{Encoded\+Input\+Stream}} or \mbox{\hyperlink{class_auto_u_t_f_input_stream}{Auto\+UTFInput\+Stream}}.

It is similar to File\+Read\+Buffer but the source is an in-\/memory buffer instead of a file.

Differences between \mbox{\hyperlink{struct_memory_stream}{Memory\+Stream}} and String\+Stream\+:
\begin{DoxyEnumerate}
\item String\+Stream has encoding but \mbox{\hyperlink{struct_memory_stream}{Memory\+Stream}} is a byte stream.
\item \mbox{\hyperlink{struct_memory_stream}{Memory\+Stream}} needs size of the source buffer and the buffer don\textquotesingle{}t need to be null terminated. String\+Stream assume null-\/terminated string as source.
\item \mbox{\hyperlink{struct_memory_stream}{Memory\+Stream}} supports Peek4() for encoding detection. String\+Stream is specified with an encoding so it should not have Peek4(). \begin{DoxyNote}{Note}
implements Stream concept 
\end{DoxyNote}
