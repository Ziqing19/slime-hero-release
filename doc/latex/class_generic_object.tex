\subsubsection*{template$<$bool Const, typename ValueT$>$\newline
class Generic\+Object$<$ Const, Value\+T $>$}

Helper class for accessing Value of object type. 

Instance of this helper class is obtained by {\ttfamily Generic\+Value\+::\+Get\+Object()}. In addition to all APIs for array type, it provides range-\/based for loop if {\ttfamily RAPIDJSON\+\_\+\+HAS\+\_\+\+CXX11\+\_\+\+RANGE\+\_\+\+FOR=1}.
