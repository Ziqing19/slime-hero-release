class Generic\+Value$<$ Encoding, Allocator $>$}

Represents a JSON value. Use Value for \mbox{\hyperlink{struct_u_t_f8}{UTF8}} encoding and default allocator. 

A JSON value can be one of 7 types. \mbox{\hyperlink{class_this}{This}} class is a variant type supporting these types.
